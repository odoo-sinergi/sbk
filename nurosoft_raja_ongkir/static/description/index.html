NUROSOFT RAJA ONGKIR

This module integrates Odoo Delivery Methods with Raja Ongkir

Dependecy Modul

  • Sales
  • Website
  • Inventory
  • Invoicing

Feature

Configuration

User can create and setting Raja Ongkir delivery method

Shipping Cost Check

User can check shipping cost according to the shipping address

Ekspedition information

Users can find out the expedition chosen by the customer through a sales order

  1. Configuration Raja Ongkir Delivery Method
    • Configuration can be accessed on the Sales > Configuration > Shipping Methods menu
    • In configuration menu:
      • Update district / city data using an API Key. API Key can be obtained from the rajaongkir.com by registering or creating an account.
      • Setting origin city / origin regency of delivery.
      • Setting free delivery fee.

        Image 1. Configuration setting

  2. Check the shipping cost of each expedition according to the destination city
    • Users can find out the cost of each available expedition through the e-commerce website, according to the shipping address.
    • User can choose the type of expedition service available.

      Image 2. Check shipping cost

  3. Ekspedition information in Sales Order
    • Users can find out the expedition chosen by the customer through a sales order.
    • The information displayed includes:
      • Expedition
      • Service
      • Delivery destination
      • Estimated
      • Cost
      • Weight

NOTE : If you use the Raja Ongkir Pro account you must increase the configuration of the limit_time_real and the limit_time_cpu before updating the Sub District (Kecamatan) data. Once the Sub District (Kecamatan) update process finished, you can restore the configuration.
